nooptimize

A tiny command line tool that allows you to disable JIT optimization for a given set of .NET assemblies.

View the Project on GitHub floriangreinacher/nooptimize

  • Download TAR Ball
  • View On GitHub

Usage

NoOptimize directory [/recursive] [/filter] [/preview] [/help]

  • directory Specifies in which directory .NET assemblies should be searched.
  • [/filter] Specifies a filter that controls which files should be considered.
  • [/help] Shows this help text.
  • [/preview] Specifies whether to show only a preview of the operations that will be performed.
  • [/recursive] Specifies whether the directory search should recurse to sub directories.
